# GitHubへの公開方法（初心者向け詳細ガイド）

現在、あなたのパソコン（ローカル）にあるコードを、インターネット上の「GitHub」という場所にアップロードする手順です。

## 手順0: フォルダをVSCodeで開く（超重要！）
**ここがうまくいっていない原因です！**
まず、今回作ったアプリのフォルダをVSCodeで正しく開く必要があります。

1.  私がアプリのフォルダを **あなたのデスクトップ (`Desktop/daikoku-tour`)** にコピーしました。
2.  VSCodeの **「Open Folder（フォルダを開く）」** ボタンをクリックします。
3.  デスクトップにある **`daikoku-tour`** というフォルダを選択して「開く」を押してください。
4.  左側のバーにファイルがたくさん表示されれば成功です！

## 手順1: GitHubで「箱」を作る
まず、コードを入れるための空のリポジトリ（箱）をGitHub上に作ります。

1. Webブラウザで [GitHub](https://github.com/) にログインします。
2. 画面右上の **「+」アイコン** をクリックし、**「New repository」** を選択します。
3. **Repository name**（リポジトリ名）を入力します。例: `daikoku-tour`
4. **Public**（公開）か **Private**（非公開）を選びます。
   - 誰でも見れるようにするなら Public
   - 自分だけ見れるようにするなら Private
5. ページ一番下の **「Create repository」** ボタンをクリックします。

## 手順2: リンクを取得する
リポジトリが作成されると、コマンドが表示された画面になります。
**「…or push an existing repository from the command line」** というセクションを探してください。
そこに以下のようなコマンドが表示されています（URL部分はあなた独自のものになります）。

```bash
git remote add origin https://github.com/あなたのユーザー名/daikoku-tour.git
git branch -M main
git push -u origin main
```

## 手順3: ターミナルで実行する (Mac & VSCodeの場合)

1. **ターミナルを開く**
   - VSCodeの上部メニューバーにある **「ターミナル (Terminal)」** をクリックします。
   - **「新しいターミナル (New Terminal)」** を選択します。
   - または、ショートカットキー `Ctrl` + `Shift` + `^` (または `~` か `@`) を押します。
   - 画面の下半分に新しいウィンドウ（ターミナル）が開きます。

2. **コマンドを貼り付ける**
   - GitHubの画面でコピーした3行のコマンドを、そのターミナルに貼り付けます（`Cmd` + `V`）。
   - キーボードの **Enter** キーを押して実行します。

## 手順4: Webサイトとして公開する (Vercel)

素晴らしいです！スクリーンショットを見る限り、GitHubへのアップロードは成功しています。
あとは、これを世界中に公開するだけです。

1. **Vercelにアクセス**
   - [Vercelの公式サイト](https://vercel.com/signup) にアクセスします。
   - **「Continue with GitHub」** を選んでログイン（または登録）してください。これが一番簡単です。

2. **プロジェクトを追加**
   - ログイン後、画面右側にある白いボタン **「Add New...」** をクリックし、**「Project」** を選択します。
   
3. **リポジトリをインポート**
   - **「Import Git Repository」** という画面になります。
   - 先ほどGitHubにアップロードした **`daikoku-tour`** という名前がリストにあるはずです。
   - その横にある白い **「Import」** ボタンを押してください。

4. **デプロイ（公開）**
   - 設定画面になりますが、何も変更する必要はありません。
   - そのまま青い **「Deploy」** ボタンを押してください。
   - 1分ほど待つと、花吹雪のアニメーションが出て完了です！

5. **完成！**
   - 画面に表示されるプレビュー画像や「Visit」ボタンをクリックすると、公開されたあなたのWebサイトが見れます。
   - そのURL（例: `daikoku-tour.vercel.app`）をスマホで開いたり、誰かに送ったりできます。
